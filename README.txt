Prices Only Members for Woocommerce allows you to display the prices only to registered users.


== Description ==

Prices Only Members for Woocommerce allows you to view easily and fast prices only for registered users. In addition, you can set the text and the position of the notification message that will be displayed if the user is not registered. In this way, only registered users may purchase on the site. It is based on the [WordPress Plugin Boilerplate](https://github.com/DevinVinson/WordPress-Plugin-Boilerplate) and it is compatible with Woocommerce 2.1+ and Polylang.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload `prices-only-members-for-woocommerce` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu in WordPress
4. Go to Woocommerce > Prices Only Members for change the default options
